DIE-HIRD (DOMAIN- INDEPENDENT ENGLISH HINDI LANGUAGE INTERFACE TO RELATIONAL DATABASE)

Person with no knowledge of database language may find it difficult to access the database. Many systems were developed to access the information from the database using natural language. But these systems are domain-dependent i.e. they were developed for a particular domain. I develop a system called DIE-HIRD for Hindi and English language which provides domain-independence. For developing a domain-independent system, I divide the system into two modules: Language processing module and Database module. Language processing module use Hindi Shallow Parser for analyzing and parsing the input query. I introduce a domain-identifier component in database module which identifies the domain by using knowledge base. I use three database domain and 80 queries for each database for testing the system. The system is able to correctly identify the domain from the query and translate it into SQL which is executed on the database and result is provided to the user in same language which is used for query.

LIBRARY MANAGEMENT SYSTEM

The project entitled Library Management System was intended to develop an automated solution for maintaining a college library. It provided an easy to use GUI for library staff to maintain specific information like issue and return of books, addition of new books into the library, book catalogues, Due/fine database etc. and for students to do custom searches on the availability of required books. The project used HTML for development of the GUI and Java Servlets (for processing request from server side) with MySQL as backend. Initially developed to work on standalone machine with a future scope for further enhancements like deployment in college server and support for operation over the network.
